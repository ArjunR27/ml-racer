## Installation
- ```pyenv virtualenv 3.12.5 ENV_NAME```
- ```pyenv activate ENV_NAME```
- ```pip install -r requirements.txt```

## Usage
To drive on your own:
- ```python human_play.py```
  
To train an agent:
- Update environment and training settings in ```config.py```
- ```python train_agent.py```
